What statement, post-incident¶

Usefulness¶

This template is useful for describing an occurrence and its significance / consequence. See an example of the template in use in what_statements.

Links¶

See the blog post by Martin Fowler about using “Given, When, Then” to specify the behavior of a system (presumably whether planned or existing).

Template¶

GIVEN <THIS_CONTEXT> WHEN <THIS_OTHER_THING_HAPPENS> THEN <SOMETHING_ELSE_RESULTS>

Negative interpersonal interaction¶

GIVEN <THESE_CIRCUMSTANCES> WHEN <THIS_HAPPENS> THEN <I_FEEL_THIS_WAY>
